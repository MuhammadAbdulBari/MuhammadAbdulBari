## Features

- Fully responsive layout, including a dedicated mobile navigation menu
- Scroll-triggered animations throughout the page
- Magnetic hover interactions on key elements
- A working, validated contact form wired up through EmailJS
- Filterable project showcase
- A design that leans on typography and whitespace instead of clutter

## Running It Locally

Clone the repo and install dependencies:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
